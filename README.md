# DofusCompagnon

Application web modulaire d'aide a la decision economique pour Dofus Touch.

Le corpus projet de reference se trouve dans `docs/`.

## Lancement

**Windows** : double-cliquer sur `start.bat`

**Terminal** : `npm run dev` a la racine du projet

## Acces

- Interface : http://localhost:5173
- API : http://localhost:3000

## Acces reseau local (LAN)

Remplace `localhost` par l'adresse IP de la machine hote.
